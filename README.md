# Chess Viz

A tool to visualize positional control throughout a chess match.

### Install Dependencies

`npm install`

### Local Development

`npm run start`

### Building

`npm run build`

### Publishing

**Ensure you build first!**

then...

`firebase deploy`

### Version Control

#### Pull Changes

`git pull`

#### Push Changes
`git add -A`

`git commit -m "This is my message"`

`git push origin master`
